# Custom-shoes
나이키 조던을 커스텀해보는 웹사이트 입니다.
![custom-shoes](./example.png)

<br>

## 프로그램 소개
- 나이키 조던의 색상을 다양하게 입혀볼 수 있습니다.
- blender, three.js를 연습해보기 위해 진행한 프로젝트입니다.

<br>

## 개발 환경
- Front: React.js, Three.js
- 디자인: blender

<br>

## 개선 목표
- 무늬나 문양 같은 이미지 파일도 커스텀해볼 수 있도록 할 예정.
